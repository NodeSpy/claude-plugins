Simplifies and refines Go code while preserving functionality.

Usage:
- `/go-code-simplifier:simplify` - Review recently modified code
- `/go-code-simplifier:simplify path/to/file.go` - Review specific file
- `/go-code-simplifier:simplify path/to/package` - Review entire package
